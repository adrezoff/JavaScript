--------------------------------------------Описание языка-------------------------------------------------------

{ Команда / аргументы / что деалет }


input / переменная / считывает данные с консоли

var / число / обявление перменной, знак "=" и затем число

if [выражение]( [тело] ) / логическое выражение [<=, >=, >, <, ==] / если "истинно" то даёт зайти в тело, если "ложно" выполняет алгорим дальше, пропуская тело

= / арефметическое выражение [*, +, -] / приравнивает перемнунную до знака "=" к результату арефметического выражения

jumpback / число > 0 / переносит IP (instruction pointer) на число ячеек влево

jumpforw / число > 0 / переносит IP (instruction pointer) на число ячеек вправо

print / переменнаю / выводит значение переменной в консоль

print  str: / строка / выводит строку в консоль

end / знак глобальной остановки выполнения алогитма
